# 💹 Economics Indicator Dashboard

An **interactive and advanced Economics Indicator Dashboard** built with **Python, Streamlit, and Plotly**, designed to visualize key economic metrics across countries and years.  

This project allows users to explore economic trends, compare multiple indicators, and gain insights into GDP, Inflation, Unemployment, and Interest Rates in an interactive, user-friendly dashboard. It’s suitable for learning, analysis, and showcasing in a portfolio or resume.  

---

## **Features**

- **Universal CSV Support:** Upload any CSV with `Country`, `Year`, and numeric indicators. The dashboard adapts automatically.  
- **Interactive Sidebar Filters:**  
  - Select one or more countries  
  - Choose multiple economic indicators  
  - Filter data by year range  
- **Dynamic KPIs:** Display the latest average values for selected indicators.  
- **Line Charts:** Track trends of indicators across years for multiple countries.  
- **Radar Chart:** Compare multiple indicators across selected countries.  
- **Modern, Futuristic Design:**  
  - Dark theme  
  - Interactive Plotly charts  
  - Responsive layout  

---

## **Technologies Used**

- **Python** – Programming language  
- **Streamlit** – Interactive web dashboard  
- **Plotly** – Advanced and interactive charts  
- **Pandas** – Data manipulation and filtering
